The Basics of the Web and HTML

The World Wide Web

The web is a collection of HTML documents that are intertwined through hyperlinks.The major pieces of the web include the web browser, the internet and the servers.The browser makes request via the internet to servers which use HTTP as protocol. Even though there can be documents, images, videos, all types of things online, HTML is what glues everything together. The servers repond with files that your browser displays.

HTML

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web and acts as the heart of the web. HTML documents are made up of text content which describes "what you see" and markup which describes "how it looks", references to other documents such as images and videos and also it could be made up of links to other pages.
